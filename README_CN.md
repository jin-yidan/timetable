# 时间表

一个简单的日程管理应用，支持网页版、PWA 和 macOS 桌面小组件。

[English](README.md)

## 下载

**macOS 应用：** [下载 Timetable.zip](../../releases/latest)

1. 下载并解压
2. 将 Timetable.app 拖到"应用程序"文件夹
3. 首次打开：系统设置 → 隐私与安全性 → 点击"仍要打开"

## 功能

- **时间线**：以列表或 24 小时时间块显示每日事件
- **周视图**：一览整周安排
- **待办**：按日期分组显示所有未完成任务
- **计划**：设置全年每月目标
- **重复事件**：支持每天、每周、每月重复
- **中国节假日**：自动显示法定节假日和二十四节气
- **日历同步**：导入 iCal 链接（如大学课表）
- **离线使用**：首次加载后无需联网
- **隐私保护**：所有数据仅存储在本地

## 使用方法

### 添加事件
1. 点击"Add event"
2. 输入时间（开始时间，结束时间可选）
3. 输入标题
4. 可选：添加备注、设置重复、标记为重要
5. 点击"Add"

### 管理事件
- 点击圆圈标记完成
- 点击 ⋯ 编辑或删除

### 导航
- 箭头按钮切换上一天/下一天或上一周/下一周
- 点击日期跳转到指定日期
- 标签页：Timeline（时间线）、Week（周）、Upcoming（待办）、Planner（计划）

### 导入日历
1. 点击齿轮图标
2. 粘贴 iCal 链接
3. 点击 Save

## 网页版

用浏览器打开 `index.html`，或访问 https://jin-yidan.github.io/timetable/

安装为 PWA：
- iOS：分享 → 添加到主屏幕
- Android：菜单 → 安装应用
- 桌面浏览器：点击地址栏的安装图标

## 数据存储

所有数据都存储在本地浏览器或应用中，不会上传到任何服务器。
